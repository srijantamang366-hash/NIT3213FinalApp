## Project Overview
An Android application that demonstrates API integration, user interface design, and Android development best practices.

## Features

- Dashboard Screen with RecyclerView list

## How to Build and Run
1. Clone this repository
2. Open the project in Android Studio
3. Wait for Gradle sync to complete
4. Run the app on an emulator or physical device

## Architecture
- MVVM Architecture
- Hilt Dependency Injection
- Retrofit for API calls
- Repository Pattern
